# Climate Modeling and Scenario Analysis Using MATLAB

## Overview

This repository contains a graduate-level engineering project completed as part of the ENGG 686: Climate Change Adaptation for Engineers course at the University of Calgary.

The project develops a simplified climate model in MATLAB to investigate Earth's energy balance, atmospheric CO₂ dynamics, and future climate scenarios under different emissions pathways.

## Project Highlights

- Developed a numerical climate model using MATLAB
- Performed sensitivity analysis of key climate parameters
- Simulated future CO₂ concentration and temperature scenarios
- Compared modeled and observed climate data
- Applied regression analysis and quantitative data analysis
- Prepared a technical report documenting methodology and results
